# p1p.pl p2p.pl p1j.java

p1p, p2p e p1j são programas em prolog para resolver problemas de restricoes.


## Usage

p1p
Corra o clp eclipse na terminal e compile.
[p1p.pl].

Para fornecer os problemas ao programa, guarde estes num ficheiro sem extensão
e corra:
nome do ficheiro:hello
iniciar(hello,A).
A será a lista de horas a iniciar as tarefas.

p2p
Corra o clp eclipse na terminal e compile.
[p2p.pl].

Para fornecer os problemas ao programa, guarde estes num ficheiro sem extensão
e corra:
nome do ficheiro:hello
iniciar(hello,A,B,C).

A será a lista de Equipas atribuidas as tarefas.
B hora de conclusao.
C erá a lista de horas a iniciar as tarefas.

p1j
compile o ficheiro p1j.java
javac p1j.java

Para fornecer os problemas ao programa, guarde estes num ficheiro sem extensão
e corra:
nome do ficheiro:hello
java p1j < hello

Todos os resultados serão imprimidos
